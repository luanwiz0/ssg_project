# 프로젝트 소개
- 책 구매는 물론, 판매, 공동구매, 경매, 독서모임 등 다양한 기능을 제공하는 도서 플랫폼
- 온라인 거래량의 증가에 따른 책 거래를 위한 플랫폼 개발
- 4인으로 진행한 팀 프로젝트 (2023.03.09 ~ 2023.06.18)

# 기여한 부분
### 장바구니 및 구매 기능
- 장바구니 물품 추가
- 결제 진행
- 관리자 페이지에서 주문 목록 확인
- 포인트 적립 및 등급 관리

### 독서모임 관련 기능
- 독서모임 추가
- 목록 확인
- 수정 및 삭제
- 회원 리스트 확인
